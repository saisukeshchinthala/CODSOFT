# CODSOFT Artificial Intelligence Internship

This repository contains the tasks completed as part of the CODSOFT AI Internship using HTML, CSS, and JavaScript.

---

## 📌 Tasks Completed

### 🔹 Task 1: Chatbot with Rule-Based Responses
- Developed a simple chatbot using JavaScript
- Uses predefined rules (if-else conditions)
- Responds to user inputs dynamically in browser
- Built with HTML, CSS for UI

---

### 🔹 Task 2: Tic-Tac-Toe AI
- Created a Tic-Tac-Toe game playable in browser
- Implemented basic AI logic using JavaScript
- Interactive UI with responsive design

---

### 🔹 Task 3: Recommendation System
- Built a simple recommendation system
- Suggests items based on user input/selection
- Implemented using JavaScript logic (no backend)
- Example: product/movie suggestions
